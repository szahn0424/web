이미지태그

이미지 파일을 연결해서 홈페이지에 보여주는 역할을 하는 태그

src속성에 이미지파일까지의 경로를 입력하여 작성한다

연결가능형태 = jpg,png,gif,svg

이미지태그는 일반적인 텍스트 글자와 같은 컨텐츠로 취급한다. 우리가 글자를 읽고 해석한 뒤 제목이라는 문맥 혹은 설명이라는 문맥으로 결정하듯이 이미지로 전체적인 맥락안에서 제목 또는 설명 컨텐츠로 구분될 수 있다

비디오태그

비디오파일을 연결해서 화면에 보여주는 태그

비디오파일은 브라우저와 호완성이 완벽하지 않아서 mp4파일(거의 대부분 호환가능쓰) 외 다른파일은 호환 문제 있음당

비디오태그의 마침태그가 있는 이유는 자식태그로 source태그를 구성하여 여러가지 형태의 비디오 타입을 수용할 수 있게 할려고 하는검당

비디오 태그는 자식태그 source태그를 위에서 아래로 탐색해나가면서 해당 브라우저와 호환되는 영상을 플레이함

유투브영상 홈페이지에 사용하기
